# Home Industrial Design

## Goal

Recreate the supplied `code.html` and `screen.png` home reference as the public
home for Bobinas, adapted to the existing Next.js application.

## Direction

The page uses a technical industrial catalog style: graphite surfaces, copper
accents, strict rectangular geometry, blueprint grid overlays, dense
specification cards, and monospaced technical labels.

## Scope

- Restyle the public navbar so it matches the reference header.
- Replace the current public home with the industrial hero, feature strip,
  technical categories, highlighted product specification cards, buyer portal,
  direct support CTA, and popup support.
- Restyle the public footer through the existing public layout.
- Keep public visitors free of prices and checkout/payment language.

## Notes

The database can remain empty after pre-delivery cleanup. The home uses curated
visual showcase cards to preserve the reference composition while real catalog
records are loaded later.
